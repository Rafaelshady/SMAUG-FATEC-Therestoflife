# 🎮 The Rest of Life

Jogo desenvolvido utilizando **Construct 2**, exportado para HTML5 e publicado online.

## 🚀 Jogar Agora

👉 https://rafaelshady.github.io/the-rest-of-life-construct/

## 🧠 Sobre o Projeto

Este projeto foi desenvolvido durante o curso de Jogos Digitais na FATEC Carapicuíba, com o objetivo de aplicar na prática conceitos de lógica de programação e desenvolvimento de jogos.

O objetivo do projeto foi:
- Criar mecânicas de jogo
- Trabalhar lógica e eventos
- Exportar para web (HTML5)
- Publicar online com GitHub Pages

## 🛠️ Tecnologias utilizadas

- Construct 2
- HTML5
- JavaScript
- Git & GitHub

## 💡 Aprendizados

Durante o desenvolvimento, foram trabalhados:

- Lógica de programação
- Estrutura de jogos
- Manipulação de eventos
- Deploy de projeto web
